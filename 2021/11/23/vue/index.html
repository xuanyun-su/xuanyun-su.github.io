初始Vue

  1. 想让Vue工作，就必须创建一个Vue示例，且要传入一个配置对象
  2. root容器里的代码依然符合html规范，只不过混入了一些特殊的Vue语法；
  3. root容器里的代码被称为【Vue模板】
  4. Vue实例和容器时一一对应的
  5. 真实开发中只有一个Vue示例，并且配合着组件一起使用

模板语法

Vue 模板语法有2大类：

  1. 插值语法：

    功能：用于解析标签体内容

    写法：，xxx是js表达式，且可以直接读取到data中所有属性

  2. 指令语法

    功能：用于解析标签（包括：标签属性，标签体内容、绑定事件。。。。。。）

    举例：v-bind:href=”xxx” 或 简写为 :href =”xxx” xxx同样是要写js表达式，且可以直接读取data中所有属性

    备注:Vue中有很多的指令，且形式都是：v-？？？？此处只是拿v-bind举例

数据绑定

Vue 有两种数据绑定的方式:

  1. 单向绑定（v-bind)：数据只能从data流向页面

  2. 双向绑定（v-model)：数据不仅能从data流向页面，还可以从页面流向data

    备注：

    1. 双向绑定一般都应用在表单类元素上（如：input，select类上等）
    2. v-model：value可以简写为v-model，因为v-model默认收集的就是value值

el和data的两种写法

data，el2种写法

  1. el2种写法

    1. new Vue时候配置el属性
    2. 先创建Vue示例，随后再通过vm.$mount(‘#root’)指定el的值

  2. data有两种写法

    1. 函数式

    2. 对象式
      如何选择：目前两种写法都可以，以后学习到组件时，data必须使用函数式，否则会报错

  3. 一个重要的原则：
    由Vue管理的函数，一定不要写箭头函数，一旦写了箭头函数，this就不再是Vue实例了。
